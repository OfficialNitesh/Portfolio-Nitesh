AI-Based Fault Detection in Power Systems

2024 • Machine Learning, Signal Processing

View Code Live Demo

The Problem

Power transmission systems fail. When they do, it costs money and causes blackouts. Most systems rely on manual detection or simple threshold-based alerts, which miss subtle faults and create false positives.

What I Built

An AI system that detects and classifies faults in medium-voltage transmission lines before they escalate. Used MATLAB and Simulink to model transmission line behavior under different fault conditions (line-to-ground, line-to-line, three-phase).

Trained Random Forest and SVM classifiers on extracted features from voltage and current waveforms. The models learned to distinguish between fault types with high accuracy.

Built a Streamlit dashboard that visualizes real-time 3-phase waveforms and displays fault predictions. Engineers can see what's happening as it happens.

Technical Stack

  • Simulation: MATLAB, Simulink for power system modeling
  • ML Pipeline: Python, Scikit-learn for model training and evaluation
  • Models: Random Forest, Support Vector Machines (SVM)
  • Frontend: Streamlit for interactive dashboard
  • Data Processing: NumPy, Pandas for feature extraction

Results

The system achieved over 95% classification accuracy on test data. Response time improved significantly compared to manual detection methods. The dashboard makes it accessible to operators without ML expertise.

Role

Solo project – system design, ML pipeline, dashboard

Duration

3 months

Tech Stack

Python MATLAB Scikit-learn Streamlit Simulink

NSUTrade Campus Marketplace

2024 • Full-Stack Web Development

View Code Live Site

The Problem

Students at NSUT needed a way to buy and sell textbooks, notes, and electronics without relying on external platforms or campus bulletin boards. Existing solutions were clunky, untrusted, or not designed for campus use.

What I Built

A full-stack marketplace built with the MERN stack. Students can list items, browse by category, message sellers, and reserve products. Authentication is secure (JWT-based), and the UI is responsive across devices.

Implemented Redux Toolkit for state management to keep the frontend predictable and scalable. Real-time listing updates ensure students see current availability.

Built an admin dashboard for moderation. Admins can review flagged listings, manage users, and view analytics on platform activity.

Technical Stack

  • Frontend: React, Redux Toolkit, Tailwind CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB with Mongoose ODM
  • Auth: JWT-based authentication
  • Deployment: Vercel (frontend), hosted API

Challenges

Managing concurrent reservations was tricky. Implemented optimistic locking to prevent double-booking. Tailwind made styling fast but required discipline to keep the design system consistent.

Results

Live platform serving NSUT students. Clean UI, fast load times, functional admin tools. Learned a lot about full-stack architecture and user-focused design.

Role

Full-stack developer

Duration

4 months

Tech Stack

React Node.js MongoDB Express Redux Tailwind

Aarti Music Player

2024 • Embedded Systems, Hardware

View Code

The Problem

Needed a device that could automatically play devotional music (aarti) at scheduled times daily. Available solutions were either too expensive or lacked flexibility for custom scheduling.

What I Built

An Arduino-based music player with RTC (real-time clock) synchronization. The device plays pre-stored tracks at fixed times without manual intervention.

Added Bluetooth control using the HC-06 module so users can adjust schedules, skip tracks, or change volume remotely via a mobile terminal app.

Designed the PCB layout in KiCad for a cleaner, more compact build. The system is low-power, reliable, and easy to replicate.

Technical Stack

  • Hardware: Arduino, RTC module, HC-06 Bluetooth, audio module
  • Software: Embedded C for Arduino
  • Design: MATLAB for initial modeling, KiCad for PCB layout

Use Cases

While built for home use, the design works for school bell systems, automated announcements, or any scheduled audio playback scenario in low-power environments.

Role

Hardware design, firmware, PCB layout

Duration

2 months

Tech Stack

Arduino Embedded C KiCad MATLAB
